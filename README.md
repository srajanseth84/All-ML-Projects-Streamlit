## Run Locally

* Clone the project

```bash
  git clone https://github.com/srajanseth84/All-ML-Projects-Streamlit.git
```

* Go to the project directory

```bash
  cd All-ML-Projects-Streamlit
```
* Create venv

```bash
  python3 -m virtualenv venv 
```

* Activate the venv

```bash
  source venv/bin/activate
```

* Install dependencies

```bash
  pip install -r requirements.txt
```
* Add your API TOKEN of Hugging face from [Hugging Face ðŸ¤—](https://huggingface.co/) in Sentiment_Classification/sentiment_classification.py


* Start the server

```bash
  streamlit run app.py 
```
